# Work & Business

**Business:** La Bandeja
**Type:** Padel sport event (community-focused)
**Location:** Argentina

## Revenue Streams
- **Player tickets** — main income source; participants pay to enroll in the event
- **Sponsorships** — sponsors provide products/prizes in exchange for exposure at the event

## Financials
- Last event generated ~900,000 ARS
- Revenue varies based on: number of gameweeks, ticket price, and court rental costs paid to the host club

## Tools
- **Google Drive** — file and media storage (currently unorganized)
- **WhatsApp** — player and team communication
- **Instagram** — main social media channel
- No MCP servers connected

## Key Pain Points
- Too much time spent on manual, repetitive tasks (messages, content decisions)
- Unorganized media in Google Drive — photos and videos not sorted or scheduled for posting
- No content calendar — posting is inconsistent and reactive
